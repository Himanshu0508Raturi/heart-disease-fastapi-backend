Heart Disease Predictor

A polished demo that sends patient features to a Random Forest model (Kaggle dataset) and visualizes the prediction.

Model: Random Forest · Dataset: Kaggle Heart Disease

About

This demo uses a Random Forest classifier trained on the popular heart disease dataset from Kaggle. Fill the form below with patient data and press Predict. The site sends a POST request and displays the model's response visually with an animated confidence gauge.
